#Machine Learning Model Prediction and Evaluation App
This project is a Machine Learning web application built using Streamlit that allows users to upload a CSV dataset, select a machine learning model, generate predictions, and evaluate model performance.

The application provides:
- Dataset upload functionality
- Multiple ML model selection
- Prediction generation
- Evaluation metrics for the selected model
- Comparison of all implemented models
- Important link : http://localhost:8501/
